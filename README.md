# [FREE] CommandVPN

Бесплатная VLESS-подписка для импорта в совместимые VPN-клиенты.

## О проекте

Этот репозиторий содержит подписку с профилем [FREE] CommandVPN для быстрого добавления серверов в клиент через ссылку или файл подписки.

## Особенности

- Готовая подписка для импорта
- Поддержка VLESS-конфигураций
- Удобное имя профиля
- Подходит для регулярного обновления списка серверов

## Использование

Скачайте или откройте файл подписки, затем импортируйте его в ваш VPN-клиент как subscription-профиль.

## Структура репозитория

- README.md
- subscription.txt

## Название профиля

[FREE] CommandVPN

## Обновление

При изменении списка серверов достаточно обновить файл подписки.
